# Qortex Axon / First Release V.1

This repository contains all the documentation and code to interface your Qortex Axon board with your computer. The following code is available:
1. Pyhton
  - Simple Serial 
  - GPS NMEA Parser
  - Face Detection
  
2. Matlab
  - Simple Serial
  - Simple Chart Plot
  - Write to file (CSV)
  - Write to file (TXT)
